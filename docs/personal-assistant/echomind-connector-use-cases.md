## Cross-Connector Use Cases (Multi-System Queries)

### Sales + Support + Communication
> "Give me a complete picture of our relationship with Acme Corp: all deals in Salesforce, support tickets in Zendesk, and recent emails from Gmail. Flag any risks."

### Project + Code + Communication
> "What's the status of the checkout redesign project? Show me Jira tickets, recent GitHub commits, and any Slack discussions about blockers."

### HR + Knowledge + Communication
> "I'm preparing for a 1:1 with an underperforming team member. Find our performance improvement process in SharePoint, any relevant Slack feedback, and HR policy docs."

### Finance + Storage + Sales
> "Prepare a customer profitability analysis for BigCorp: contract value from Salesforce, invoices from Google Drive, and support costs from Zendesk."

### Product + Community + Sales
> "What features are customers asking for? Cross-reference ProductBoard requests, Discourse forum discussions, and Gong call transcripts."

## Summary: Query Patterns

| Pattern | Example |
|---------|---------|
| **Find/Search** | "Find all contracts with..." |
| **Summarize** | "Summarize the discussion about..." |
| **Status/Report** | "What's the status of..." |
| **Compare** | "Compare our performance in Q1 vs Q2..." |
| **List** | "List all tickets assigned to..." |
| **Explain** | "Explain how the pricing module works..." |
| **Cross-reference** | "Cross-reference customer X across all systems..." |
| **Alert/Monitor** | "Alert me when tickets breach SLA..." |
| **Generate** | "Generate a weekly report of..." |
| **History** | "What's the history of our interactions with..." |
